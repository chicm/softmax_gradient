Let's say we have a nueron network with softmax classifier at the last layer, using cross entropy loss function.

Softmax function is defined as following:

Cross entropy loss function is defined as following:

L=−∑iyilog(pi) L = -\\displaystyle\\sum\_{i} y\_ilog(p\_i) L=−i∑​yi​log(pi​)

Assume the index of correct class is k (here in this exmple, k=1), for incorrect class, the yiy\_iyi​ is always 0, therefore, the loss function can be simplified as:
